# Prompts Library

Delt bibliotek med gjenbrukbare prompt-snutter, komponenter og mønstre som kan inngå i templates. Dette er felles byggeklosser, ikke ferdige template-prompter.

Rolle i templatesystemet:
- `_library/prompts` er single-source-of-truth for delte prompt-komponenter.
- Per-template prompts plasseres i `gpt-packages/templates/<template>/prompts`.

Hva som finnes her:
- Snippets, mønstre og små include-utterances som er enkle å komponere.
